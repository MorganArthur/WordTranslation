# 屏幕翻译助手

屏幕翻译助手是一个Chrome扩展程序，可以帮助用户选择屏幕上的任何区域（包括图片、视频等无法直接选中的内容），识别其中的英文文字并翻译成中文。

## 功能特点

- 屏幕区域选择：可以选择网页上的任何区域，包括图片、视频等无法直接选中的内容。
- OCR文字识别：使用Tesseract.js进行准确的文字识别。
- 实时翻译：将识别出的英文文字翻译成中文。
- 简洁的用户界面：易于使用，结果显示清晰。

## 安装步骤

1. 下载本项目的ZIP文件或克隆仓库到本地。
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）。
3. 在右上角打开"开发者模式"。
4. 点击"加载已解压的扩展程序"。
5. 选择项目文件夹。

## 配置API密钥

在使用屏幕翻译助手之前，您需要配置阿里云翻译服务的API密钥：

1. **获取阿里云API密钥**
   - 登录 [阿里云RAM控制台](https://ram.console.aliyun.com/manage/ak)
   - 创建或查看AccessKey，获取`AccessKeyId`和`AccessKeySecret`
   - 确保账户已开通机器翻译服务并有足够余额

2. **配置密钥**
   - 点击扩展图标打开弹出窗口
   - 点击右上角的⚙️设置按钮或点击提示中的"配置翻译API"
   - 在配置页面输入您的`AccessKeyId`和`AccessKeySecret`
   - 点击"保存配置"按钮
   - 可以点击"测试连接"验证配置是否正确

## 使用方法

1. 确保已正确配置API密钥（首次使用会有提示）。
2. 点击Chrome工具栏中的扩展图标打开弹出窗口。
3. 点击"开始截图翻译"按钮（需要先配置API密钥）。
4. 在网页上拖动鼠标选择要翻译的区域。
5. 等待文字识别和翻译完成。
6. 查看翻译结果。

## 注意事项

- **首次使用**：必须先配置阿里云API密钥才能使用翻译功能。
- **API费用**：翻译服务会产生费用，请根据阿里云的计费标准进行预估。
- **文字识别**：确保选择的区域包含清晰可辨的英文文字。
- **网络要求**：翻译功能需要网络连接访问阿里云服务。
- **隐私安全**：API密钥存储在本地浏览器中，不会上传到任何第三方服务器。
- 如遇到问题，请检查API密钥配置或刷新页面后重试。

## 隐私声明

- 本扩展程序不会收集或存储任何个人信息。
- OCR文字识别使用Tesseract.js在本地完成，不涉及数据传输。
- 翻译功能使用阿里云翻译服务，识别的文字会发送到阿里云进行翻译。
- API密钥仅存储在您的浏览器本地存储中，不会传输给任何第三方。
- 建议定期更换API密钥以确保账户安全。
