# 屏幕翻译助手

屏幕翻译助手是一个Chrome扩展程序，可以帮助用户选择屏幕上的任何区域（包括图片、视频等无法直接选中的内容），识别其中的英文文字并翻译成中文。

## 功能特点

- 屏幕区域选择：可以选择网页上的任何区域，包括图片、视频等无法直接选中的内容。
- OCR文字识别：使用Tesseract.js进行准确的文字识别。
- 实时翻译：将识别出的英文文字翻译成中文。
- 简洁的用户界面：易于使用，结果显示清晰。

## 安装步骤

1. 下载本项目的ZIP文件或克隆仓库到本地。
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）。
3. 在右上角打开"开发者模式"。
4. 点击"加载已解压的扩展程序"。
5. 选择项目文件夹。

## 使用方法

1. 点击Chrome工具栏中的扩展图标打开弹出窗口。
2. 点击"开始截图翻译"按钮。
3. 在网页上拖动鼠标选择要翻译的区域。
4. 等待文字识别和翻译完成。
5. 查看翻译结果。

## 注意事项

- 确保选择的区域包含清晰可辨的英文文字。
- 翻译结果的准确性可能受到OCR识别质量和网络状况的影响。
- 如遇到问题，请刷新页面后重试。

## 隐私声明

本扩展程序不会收集或存储任何个人信息。所有的文字识别和翻译操作都在本地完成。

## 技术栈

- JavaScript
- HTML/CSS
- Chrome Extension API
- Tesseract.js (OCR)
- 免费的在线翻译API
